## 🎯 功能特点

- **一键启动**：按下 PA0 按键，蜂鸣器提示，OLED 显示启动信息。
- **实时显示**：OLED 显示当前步骤、超声波测距（cm）和红外模块状态。
- **双重检测**：超声波（距离 <12cm 或 >120cm 视为无效） + 红外（低电平表示障碍物），可靠触发避障。
- **分段教学演示**（共 5 次触发）：
  1. 直走 → 遇障 → 等待移除 → 舵机扫描（左/中/右/右/中） → 左偏斜走 → 回正。
  2. 直走 → 遇障 → 等待移除 → 舵机扫描 → 右偏斜走 → 回正。
  3. 直走 → 遇障 → 等待移除 → 左偏斜走 → 回正。
  4. 最终模式：无限循环，无障碍则前进，有障碍则停车。
- **舵机扫描**：通过 PWM 控制舵机左右转动，实现超声波测距的扫描动作。
- **蜂鸣器提示**：每次检测到障碍物并停车时，蜂鸣器短鸣一次。

## 🔌 硬件接线

| 外设         | PyBoard / STM32 引脚   | 备注                         |
|--------------|------------------------|------------------------------|
| 右前电机     | PB6 (PWM)              | 通道1，占空比控制转速        |
| 左前电机     | PB7 (PWM)              | 通道2                        |
| 左后电机     | PB8 (PWM)              | 通道3                        |
| 右后电机     | PB9 (PWM)              | 通道4                        |
| 超声波 Trig  | PB3 (输出)             | 触发信号                     |
| 超声波 Echo  | PB4 (输入，下拉)       | 接收回波，测量脉冲宽度       |
| 红外避障模块 | PB5 (输入)             | 低电平=有障碍（灯亮）        |
| 舵机         | PB10 (PWM)             | 50Hz 周期，7.5% 占空比中位  |
| OLED (I2C)   | PB12 (SDA), PB13 (SCL) | 地址 0x3C，128×64 像素       |
| 蜂鸣器       | PA8 (PWM)              | 无源蜂鸣器，PWM 驱动         |
| 启动按键     | PA0 (上拉输入，中断)   | 下降沿触发启动               |

> **电机驱动**：建议使用 L298N 或 TB6612 驱动板，引脚输出 PWM 信号控制使能与方向。代码中 `pulse_width_percent` 为 30% 占空比前进，0 为停止。

## 🧩 外设说明

- **超声波模块 (HC-SR04)**  
  测量距离 2~120cm，超时返回 999（视为无效）。测量周期不宜过密，代码中主循环每 100ms 读取一次。

- **红外避障模块**  
  输出信号：灯亮 → 障碍物 → 输出低电平(0)；灯灭 → 无障碍 → 输出高电平(1)。代码中 `ir_has_obstacle()` 返回 True 表示有障碍。

- **舵机 (SG90 或类似)**  
  PWM 频率 50Hz，占空比范围 5%~10% 对应 -90° 到 +90°。代码中定义：
  - 中位 (0°)：7.5% 占空比
  - 左转 90°：7.3% 占空比（需根据实际校准）
  - 右转 90°：7.7% 占空比

- **电机动作函数**：
  - `go()`：四轮 30% 占空比前进
  - `stop()`：所有通道占空比 0
  - `turn_left_bias()`：左侧两轮 30%，右侧 0 → 向左前方斜走
  - `turn_right_bias()`：右侧两轮 30%，左侧 0 → 向右前方斜走
  - `go_straight()`：同 `go()`

    Straight1 --> Step2[阶段2: 直走]
    Step2 --> Check2{超声波<12cm 或 红外有障碍?}
    Check2 -- 否 --> Step2
    Check2 -- 是 --> Stop2[停车+蜂鸣] --> WaitRemove2[等待障碍物移除] --> Scan2[舵机扫描一次] --> RightBias[右偏斜走 630ms] --> Straight2[回正直走]
